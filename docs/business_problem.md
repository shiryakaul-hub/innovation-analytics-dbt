
# Business Problem

Organisations often manage innovation initiatives through a project lifecycle.

While source systems provide the current status of each initiative, they do not retain historical changes.

This makes it difficult to answer:

- How many initiatives progressed this month?
- Which initiatives regressed?
- How long do initiatives remain in each stage?
- What was the portfolio value six months ago?
- Initiatives can contribute to anything ranging from KPIs to metrics related to performance(features)

This project demonstrates how dbt snapshots and dimensional modelling can be used to enable historical reporting and trend analysis.
